# Vim-Web

> ⌨️ **键盘即世界**：让 Chrome 浏览体验回归 Vim 的纯粹与高效。

Vim-Web 是一款轻量级的 Chrome 浏览器扩展，旨在通过 Vim 风格的快捷键重构人与网页的交互方式。它不依赖 Vue/React 等重型框架，追求极致的性能与原生 DOM 控制力。

## ✨ 特性 (Features)

*   **⚡️ 极致轻量**：核心逻辑仅使用原生 JavaScript，无外部依赖，秒级启动。
*   **🛡️ 输入保护**：智能检测输入框 (Input/Textarea)，在打字时自动暂停接管，互不干扰。
*   **🖱️ 平滑滚动**：基于屏幕比例的智能滚动算法，浏览体验流畅自然。
*   **🔧 开箱即用**：无需繁琐配置，安装即可使用最经典的 Vim 键位。

## 🚀 安装指南 (Installation)

1.  克隆或下载本项目到本地。
2.  打开 Chrome 浏览器，访问 `chrome://extensions/`。
3.  开启右上角的 **"开发者模式" (Developer mode)**。
4.  点击左上角的 **"加载已解压的扩展程序" (Load unpacked)**。
5.  选择本项目的根目录 (`wim/`) 即可。

## ⌨️ 快捷键 (Shortcuts)

### 导航 (Navigation)
| 按键 | 功能 | 说明 |
| :--- | :--- | :--- |
| `j` | 向下滚动 | 滚动自定义步长（默认 15% 屏幕高度） |
| `k` | 向上滚动 | 滚动自定义步长（默认 15% 屏幕高度） |
| `h` | 向左滚动 | 滚动自定义步长 |
| `l` | 向右滚动 | 滚动自定义步长 |
| `gg` | 回到顶部 | 快速跳转至页面最上方 |
| `G` | 直达底部 | 快速跳转至页面最下方 |
| `Ctrl+d` | 向下半页 | 滚动 50% 屏幕高度 |
| `Ctrl+u` | 向上半页 | 滚动 50% 屏幕高度 |

### 页面交互 (Interaction)
| 按键 | 功能 | 说明 |
| :--- | :--- | :--- |
| `f` | Hint 模式 | 为页面可点击元素生成字母标签，按对应字母即可点击 |
| `Space` | 点击光标处 | 点击鼠标指针所在位置的元素 |
| `q` / `Q` | 返回上一页 | 相当于浏览器后退按钮 |
| `Esc` | 退出/复位 | 退出 Hint 模式或 Insert 模式，返回 Normal 模式 |

### 模式说明 (Modes)
- **NORMAL 模式**：默认模式，可使用所有快捷键
- **INSERT 模式**：在输入框中自动进入，快捷键不生效
- **HINT 模式**：按 `f` 进入，用于键盘点击页面元素

## ⚙️ 设置 (Settings)

右键点击扩展图标 → **选项**，可配置：
- **滚动步长**：自定义 j/k/h/l 键的滚动距离（百分比或像素）
- **黑名单**：在指定域名上禁用插件（支持通配符，如 `*.example.com`）
