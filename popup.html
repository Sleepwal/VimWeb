Vim Web

v0.5

导航

  • j 向下滚动
  • k 向上滚动
  • h 向左滚动
  • l 向右滚动
  • gg 回到顶部
  • G 直达底部
  • Ctrl+d 向下半页
  • Ctrl+u 向上半页

搜索

  • / 打开搜索
  • n 下一个匹配
  • N 上一个匹配
  • * 搜索光标下单词

交互

  • f Hint 模式
  • Space 点击光标处
  • q / Q 返回上一页
  • Esc 退出模式

跳转

  • gi 上一个输入框
  • gI 第一个输入框
  • ]] 下一页链接
  • [[ 上一页链接

标签页

  • gt 下一个标签页
  • gT 上一个标签页
  • b 标签页列表
  • x 关闭当前标签页
  • X 恢复关闭的标签页

书签与历史

  • B 打开书签
  • H 打开历史记录
